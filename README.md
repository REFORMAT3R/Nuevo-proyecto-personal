# Smash Hub

## URL del proyecto
https://nuevo-proyecto-personal.onrender.com

## Descripción
Smash Hub es una aplicación web desarrollada con Flask que permite visualizar
torneos de Super Smash Bros Ultimate, incluyendo torneos próximos, torneos
finalizados y una sección de jugadores destacados.  
El proyecto fue creado con fines académicos y utiliza datos de ejemplo.

## Tecnologías utilizadas
- **HTML**: estructura de las páginas web  
- **CSS**: estilos y diseño visual  
- **JavaScript**: manejo de contenido dinámico  
- **Python (Flask)**: backend y sistema de rutas  
- **SQL**: manejo de datos de ejemplo
